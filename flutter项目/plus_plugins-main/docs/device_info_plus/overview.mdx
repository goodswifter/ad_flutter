## Device Info Plus Overview

Get current device information from within the Flutter application.

### Get started

#### 1. Add it to your project

```yaml {4} title="pubspec.yaml"
dependencies:
  flutter:
    sdk: flutter
  device_info_plus: "^{{ plugins.device_info_plus }}"
```

#### 2. Download dependency

```bash
$ flutter pub get
```

#### 3. Rebuild the app

```bash
$ flutter run
```
